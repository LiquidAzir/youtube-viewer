About this app
Search is powered by the free YouTube Data API v3. The shared key gives everyone ~100 searches/day combined. For your own ~100/day:
1. Visit console.cloud.google.com
2. Create a project → enable YouTube Data API v3
3. Credentials → Create credentials → API key
4. Paste it in Settings (gear icon, top right)

Settings

Personal API key (optional)
3-minute setup for your own ~100 daily searches:
1. Open console.cloud.google.com
2. Create a project (any name)
3. APIs & Services → Enable → YouTube Data API v3
4. Credentials → Create credentials → API key
5. Copy AIza… → paste above → Save
Restrict the key to YouTube Data API v3 after creating it.
My YouTube channel (optional)
Shows your uploads + public playlists in the My YouTube tab.
1. Find your ID at youtube.com/account_advanced (starts with UC)
2. Paste above → Save
To see your subscriptions: YouTube → Settings → Privacy → turn off “Keep all my subscriptions private.”
Watch Later playlist (optional)
Your own “watch later” queue — the Watch Later button on the home screen opens it.
1. On YouTube, make a playlist (e.g. “Glasses”), set it to Unlisted
2. Open the playlist; copy the PL… from the URL (…list=PL…)
3. Paste above → Save
On your computer, click Save under any video → that playlist to add it.
